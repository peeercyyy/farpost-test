# Тестовое задание для FarPost

## Запуск проекта в режиме разработки

### 1. В корневой директории 
   ```
   npm install
   npm run dev
   cd frontend
   npm run start
   ```
  Откроется окно на <localhost:3000>

## Запуск проекта в режиме продакшена

### 1. В корневой директории
  ```
  npm install
  npm run build
  npm run start
  ```
  Проект будет доступен на <localhost:3001>
